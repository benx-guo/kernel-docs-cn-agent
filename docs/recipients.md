# Patch Recipients

## Required Recipients

All translation patches must be sent to the following addresses:

| Role                            | Address                          |
|---------------------------------|----------------------------------|
| Chinese documentation maintainer| Alex Shi <alexs@kernel.org>      |
| Documentation mailing list      | linux-doc@vger.kernel.org        |

### Role Descriptions

- **Alex Shi** is the maintainer of the Chinese kernel documentation
  translations. All Chinese translation patches are reviewed and applied
  by this maintainer.
- **linux-doc@vger.kernel.org** is the kernel documentation mailing list
  where all documentation patches (including translations) are discussed
  and archived publicly.

## Internal Review Address

For the internal review stage (`--review`), the reviewer's email address is
configured by the user in `config/email.conf`. This address is not published
and is specific to each contributor's review workflow.
